# Numeric Property

All numeric properties in Pal Schema are treated the same. `Pal/Content/Pal/DataTable/Character/DT_PalMonsterParameter.uasset` as an example:

```json
"Rarity": 4,
"EnemyMaxHPRate": 1.0
```

`Rarity` is an integer type while `EnemyMaxHPRate` is a float.
